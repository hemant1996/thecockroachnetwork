Sawal-jawab

Kya yeh free hai?

Haan. Hamesha. Code CC0 — public domain. Koi paid plan, koi ads, koi pay-to-win. Server ka kharcha jo run karta hai woh dekhta hai, aapko kuch nahi.

Mera data leak ho jayega?

Aapki gupt chaabi sirf phone mein hai. Kahin server par nahi. Password reset nahi hota. Phone gaya = pehchaan gayi — yeh tradeoff hai. (Backup feature aa raha hai.)

Naam-pata nahi dete, sirf ek random key hoti hai. Lekin jo report karte ho woh public hai — woh toh point hai.

Kya police mujhe identify kar sakti hai?

Honest baat: relay se connect karte ho, relay aapka IP dekh sakta hai. Sensitive cheez likhni ho to Tor browser ya VPN se kholo. Location bhi dhundli rakh — precision 5 ya kam.

Phone-key se asli naam connect nahi hai. Pseudonymous ho — pakka anonymous nahi. Soch ke daal.

Kya sirf Cockroach Janta Party walon ke liye hai?

Nahi. Yeh ek protocol hai, kisi ka private nahi. Koi bhi use kar sakta hai — citizens, journalists, activists, students, jo bhi. CJP ne yeh moment banaya, tool sab ka hai.

Kya phone storage chahiye?

Bas browser. Koi app nahi download karna. thecockroachnetwork.com kholo, bas.

Kahan hosted hai? Kya band ho sakta hai?

Code GitHub par. Pehla relay Mumbai mein. Aur log bhi relay chala sakte hain — agar ek band ho, doosra chalega. Aapka phone bhi ek relay hai (peer mode). Pura network band karna asambhav.

Mujhe code samajhna hai / fork karna hai?

Saari cheez open hai. Developer page dekh — spec, source, run-your-own. Apna relay chalao, apna mirror host karo, naya kuch banao.
