Become a Cockroach Pehredaar

The truth: there is only one public Pehredaar today, in Mumbai. If it goes down and nobody else has set one up, everything you have written becomes unreachable. The network dies.
That is why Pehredaars are needed. The more there are, the stronger the network.

What does a Pehredaar do?

Keeps a server running. That's it. The server receives every report, stores it, forwards it to others. No code knowledge needed.

3 minutes to become a Pehredaar — no credit card

  1. Click the Deploy to Render button below.
  2. Sign in or create a Render.com account — free, no card.
  3. Name your Pehredaar, like cockroach-relay-yourname.
  4. Click "Apply". Render deploys it automatically.
  5. After ~3 minutes you get a URL: https://[yourname].onrender.com.
  6. Share that URL — Twitter, WhatsApp, anywhere. Other clients will use your relay.

Sawaal aur jawaab

Kya yeh free hai?

Haan, hamesha. Code CC0 — public domain. Koi paid plan nahi, koi ads nahi. Server cost jo run karta hai woh personally afford karta hai — aapko kuch nahi dena.

Kya mera data leak ho sakta hai?

Aapki gupt chaabi sirf aapke device ke browser mein hai — kisi server par nahi. Password reset nahi hota. Device badla ya browser data clear hua = pehchaan gayi. Yeh ek tradeoff hai — encrypted backup feature v0.2 mein aa raha hai.

Aap apna naam-pata nahi dete, sirf ek random key hoti hai. Lekin jo report karte hain woh public hai — woh toh point hi hai.

Kya police mujhe identify kar sakti hai?

Honest answer: aap relay se connect karte hain, relay aapka IP dekh sakta hai. Sensitive reports ke liye Tor browser ya VPN se kholiye. Location precision bhi kam rakhiye (5 ya kam).

Phone-key se asli naam directly connect nahi hai. Pseudonymous hain — pakka anonymous nahi. Soch ke daaliye.

Kya sirf Cockroach Janta Party walon ke liye hai?

Nahi. Yeh ek protocol hai, kisi ka private nahi. Koi bhi use kar sakta hai — citizens, journalists, activists, students, jo bhi. CJP movement ne yeh moment banaya; tool sab ka hai.

Kuch download karna hoga?

Nahi. Bas browser. Phone ho ya laptop, koi app install nahi. thecockroachnetwork.com kholiye — bas.

Agar saare Pehredaar band ho gaye to?

Network mein jo events the, woh khatam ho jaayenge — kyunki host karne wala koi nahi. Aapki gupt chaabi safe hai (woh aapke browser mein), lekin reports khoja sakte hain. Yahi reason hai ki Pehredaar chahiye. Aap khud Pehredaar baniye.

Mujhe code dekhna hai / fork karna hai?

Questions

Is this free?

Yes, always. Code is CC0 — public domain. No paid plans, no ads. Whoever runs a server personally affords the cost — you pay nothing.

Can my data leak?

Your secret key stays only in your device's browser — never on a server. There is no password reset. Change device or clear browser data = identity lost. This is the tradeoff — an encrypted backup feature is coming in v0.2.

You don't provide a real name, just a random key. But what you report is public — that's the point.

Can the police identify me?

Honest answer: you connect to a relay, and the relay can see your IP. For sensitive reports, open it via Tor browser or a VPN. Keep the location precision low (5 or less).

Your phone key is not directly linked to your real name. You are pseudonymous — not fully anonymous. Think before you post.

Is this only for Cockroach Janta Party members?

No. This is a protocol, not anyone's private tool. Anyone can use it — citizens, journalists, activists, students, anyone. The CJP movement created this moment; the tool belongs to everyone.

Do I need to download anything?

No. Just a browser. Phone or laptop, no app to install. Open thecockroachnetwork.com — that's it.

What if every Pehredaar goes offline?

Every event in the network becomes unreachable — nobody is hosting it anymore. Your secret key is safe (it lives in your browser) but the reports are lost. That is why Pehredaars matter. Become one yourself.

I want to read the code / fork it?

Everything is open. See the developer page — spec, source, run-your-own guides. Run your own relay, host your own mirror, build something new on top.
