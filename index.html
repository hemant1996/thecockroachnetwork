Sawal-jawab

Kya yeh free hai?

Haan. Hamesha. CC0 public domain — code, spec, sab kuch. Koi paid plan nahi, koi ads nahi. Server cost koi operator personally afford karta hai; aapko kuch nahi dena.

Mera data leak ho jayega?

Aapki gupt chaabi (secret key) sirf aapke phone par hai. Kisi server par nahi. Koi password reset nahi. Phone kho gaya = pehchaan kho gayi (v0.2 mein backup add hoga).

Aap apni pehchaan ka naam-pata nahi dete — sirf ek random key hoti hai. Lekin aap jo report karte ho woh public hai (woh toh point hai).

Kya police mujhe identify kar sakti hai?

Honest answer: aap relay se connect karte ho, relay aapka IP dekhta hai. Sensitive reports ke liye Tor browser use karo, ya VPN. Location ko bhi coarse rakho (5 ya kam precision).

Aapka phone-key se asli naam connect nahi hai — pseudonymous ho. Lekin pakka anonymous nahi. Decide kar ke report karo.

Kya sirf Cockroach Janta Party walon ke liye hai?

Nahi. Yeh ek protocol hai. Koi bhi use kar sakta hai — citizens, journalists, activists, anyone. CJP movement ne is moment ko create kiya, lekin tool sab ka hai.

Kya phone storage chahiye?

Bas browser. Koi app download nahi karna. Open thecockroachnetwork.com on your phone — bas.

Kahan hosted hai? Kya band ho sakta hai?

Code GitHub par hai (CC0). Pehla relay Fly.io Mumbai mein. Aur relays bhi anyone run kar sakta hai — agar pehla band ho, doosra chalega. Aapka phone bhi ek relay hai (peer mode). Network kabhi puri tarah band nahi ho sakta.

Mujhe code samajhna hai / fork karna hai?

Sure — yeh sab open hai. Developer page dekho — spec, source, run-your-own guides. Anyone can run their own relay, host their own client mirror, build new tools on top.
