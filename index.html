Sawaal aur jawaabQuestions & answers

Sawaal hain?
Ek baar padh leejiye. Questions?
Read these first.

Kya yeh free hai?Is it free?

Haan, hamesha. Code CC0 — public domain. Koi paid plan, koi ads. Server cost personally afford hoti hai — aapko kuch nahi dena. Yes, always. Code is CC0 — public domain. No paid plans, no ads. Server cost is personally afforded — you pay nothing.

Mera data leak ho sakta hai?Can my data leak?

Aapki gupt chaabi sirf aapke device ke browser mein hai — kisi server par nahi. Device badla = pehchaan gayi. Identity tab se export kar sakte hain. Your secret key lives only in your browser — never on a server. Change device = identity is gone unless you export it from the Identity tab.

Police identify kar sakti hai?Can the police identify me?

Aap relay se connect karte hain, relay aapka IP dekh sakta hai. Sensitive ho to Tor ya VPN se kholiye. Location precision kam rakhiye. You connect to a relay; the relay can see your IP. For sensitive reports use Tor or a VPN. Keep location precision coarse.

Sirf CJP walon ke liye?Only for the CJP movement?

Nahi. Yeh ek protocol hai, kisi ka private nahi. Koi bhi use kar sakta hai — citizens, journalists, activists, students. No. This is a protocol, nobody's property. Anyone can use it — citizens, journalists, activists, students.

Kuch download karna hoga?Do I download anything?

Nahi. Bas browser. Phone ya laptop, koi app install nahi. Site kholiye — bas. No. Just the browser. Phone or laptop, no app to install. Open the site — that's it.

Sab Pehredaar band ho gaye to?What if all relays go down?

Reports kho ja sakte hain. Aapki gupt chaabi safe hai. Isi liye Pehredaar chahiye. Reports could be lost. Your secret key is safe. That's why we need more Pehredaar.
